# D•Tunes Music Player

A responsive music player built with HTML, CSS, and JavaScript.

## Live Demo

🔗 https://yourusername.github.io/d-tunes/

## Features

- Play/Pause Music
- Next & Previous Track
- Shuffle & Repeat
- Favorites using LocalStorage
- Search Songs
- Queue Management
- Keyboard Shortcuts
- Responsive Design

## Technologies Used

- HTML5
- CSS3
- JavaScript


## Installation

1. Clone the repository
2. Open index.html in your browser
